## Local Development

```bash
# Setup
npm install

# Run
npm start

# Test
npm test

# Build
npm run build
```

## Key Features
- React 18 with Tailwind CSS
- Mobile-first responsive design
- Real-time chat interface
- Custom dog/human/companion icons
